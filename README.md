Bla# Трекер задач МКУ ЦБО

Мобильное приложение для создания и отслеживания задач для Муниципального казенного учреждения города Казани "Центр бухгалтерского обслуживания" (МКУ ЦБО).

## Функциональность

- Создание и редактирование задач
- Установка сроков выполнения
- Отслеживание прогресса (0-100%)
- Установка приоритетов (Низкий, Средний, Высокий, Критический)
- Фильтрация задач по статусу
- История изменений задач
- Push-уведомления о приближающихся сроках
- Экспорт задач в CSV
- Валидация данных
- JWT аутентификация

## Технологический стек

### Мобильное приложение
- **React Native** с **Expo** (SDK 56)
- **TypeScript**
- **Expo Router** (файловая навигация)
- **Zustand** (управление состоянием)
- **React Query** (серверное состояние)
- **Axios** (HTTP-клиент)
- **Expo Notifications** (push-уведомления)

### Backend
- **Node.js** + **Express** (TypeScript)
- **Drizzle ORM** + **PostgreSQL**
- **JWT** аутентификация
- **Zod** валидация
- **node-cron** (планировщик уведомлений)
- **Expo Server SDK** (push-уведомления)

### Инфраструктура
- **Docker** + **Docker Compose**
- **PostgreSQL 16**

## Установка и запуск

### Требования
- Node.js 20+
- Docker Desktop
- Expo CLI (`npm install -g expo-cli`)


### 1. Запуск Backend (через Docker)
```bash
docker-compose up -d
```
Backend будет доступен на `http://localhost:3001`

### 2. Запуск мобильного приложения
```bash
cd apps/mobile
npm install
npx expo start
```

### Переменные окружения (Backend)
Создайте файл `apps/backend/.env`:
```
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/mku_cbo_task_tracker"
JWT_SECRET="ваш-секретный-ключ"
JWT_EXPIRES_IN="7d"
PORT=3001
EXPO_ACCESS_TOKEN="ваш-expo-токен"
```

## API Endpoints

### Аутентификация
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/auth/register` | Регистрация |
| POST | `/api/auth/login` | Вход |
| GET | `/api/auth/profile` | Профиль (требует JWT) |

### Задачи
| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/tasks` | Список задач (с фильтрацией) |
| POST | `/api/tasks` | Создание задачи |
| GET | `/api/tasks/:id` | Детали задачи |
| PUT | `/api/tasks/:id` | Обновление задачи |
| DELETE | `/api/tasks/:id` | Удаление задачи |
| GET | `/api/tasks/:id/history` | История изменений |

### Экспорт
| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/export/csv` | Экспорт задач в CSV |

### Уведомления
| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/push-token` | Регистрация push-токена |
| POST | `/api/notifications/test` | Тестовое уведомление |
